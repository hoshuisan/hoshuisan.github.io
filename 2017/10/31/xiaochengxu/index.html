浅谈微信小程序

发表于 2017-10-31

微信小程序已经上线10个月，在这十个月中微信小程序已经更新至1.01版本。我也从6月份开始一直制作微信小程序，至今已经成功上线了两款小程序，这篇文章主要说一说我自己对微信小程序的理解和看法，如果有一些错误的地方，欢迎大家批评指导。

一、小程序概述

1.定义

微信官方对小程序的定义是——一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验。而张小龙先生对微信小程序是这样描述的—— “小程序是一种不需要下载安装即可使用的应用，体现了用完即走的理念。微信是个工具，是个非常宏大的目标”。微信官方将微信小程序定位为体验比网页好，比下载App更便捷，具有无需安装、触手可及、用完即走、无需卸载四个特性的软件。

2.流量入口

(1).搜索

用户直接在小程序页面搜索关键词，或者在“搜一搜”功能中搜索关键词。这个入口并不是目前小程序的重要入口。由于小程序和“搜一搜”的普及度不高，比较少用户会通过搜索进入小程序。

(2).微信公众号

微信公众号是微信生态最重要的组成部分。一个公众号能绑定50个微信小程序，许多公众号也已经悄悄绑定了自己的小程序。用户能通过微信公众号的菜单、文章、介绍页面进入小程序。

(3).用户分享

用户分享能分为两种方式。第一种是用户直接将小程序分享到会话页面，这是微信官方允许的，在小程序功能中能轻松实现。第二种是开发者钻一下微信的漏洞，为用户生成小程序的二维码供用户保存，以便用户分享到朋友圈。由于小程序并没有分享到朋友圈的功能，因此用户并不能直接分享到朋友圈。但是朋友圈爆款是小程序获取流量的重要手段之一，许多社交类小程序就使用了二维码分享的方法，引导用户保存二维码并将二维码分享到朋友圈。

(4).附近的小程序

根据地理位置在小程序页面推荐小程序，这个入口主要以进入线下门店小程序为主，让小程序更好地服务线下门店。

3.功能

我这里所说的小程序的功能是指微信对小程序的设计、开发、运营的规定下小程序所能实现的能力。小程序的能力由微信官方设计，但是特定小程序的功能却是由开发者们制作的。小程序的发展决定于小程序对开发者的开放程度。目前小程序能实现发起／接受网络请求；小程序分享；支付；实时通信；调用系统工具比如相机、录音、手机通讯录、读取／保存文件、读取／保存图片、播放音乐／视频、蓝牙等手机相关功能；地理位置定位等功能。同时最新更新了小程序承载网页的功能，能在小程序中直接打开网页，也能从网页中直接回到小程序中。小程序调用手机功能的能力已经相当完善了，手机技术中比较新的生物识别功能也可以在小程序中进行调用，但是小程序与微信生态的相关功能并没有开放给开发者们。小程序并不能向公众号文章一样分享到朋友圈，也不能从小程序回到公众号。在我看来，小程序在微信生态圈中属于一个分支，并没有与其他微信生态的功能形成一个闭环。

二、小程序产品分析

1.小程序+社交

案例

群印象

  • 主要功能：自己设置好友印象，分享给好友，邀请好友选择对自己的印象，达到社交功能。

分析：

  • 群印象通过生成用户专用二维码引导用户将小程序分享到朋友圈，达到了朋友圈爆款的效果，但是由于微信官方不允许小程序分享到朋友圈，因此微信很快将群印象进行了封禁。随着小程序分享到微信群的功能开放，小程序对分享功能的开放也会逐渐提高，但是我认为微信本身就是一个社交软件，小程序+社交只能成为朋友圈的爆款，并不能长期运营。

2.小程序+线下

案例

肯德基+

  • 主要功能：点餐、领取优惠券、使用积分

分析：

  • 微信通过小程序打通了微信支付与b端的连接，让中小企业能通过微信小程序实现点餐，收款等功能。我认为小程序将来发展一个很重要的方向就是连接线下商户。在线下场景中，小程序有着极大提高用户体验的空间，它能提高商户的服务质量，降低人力成本，如手机点餐、支付，也能提高顾客的用户体验，节省时间，如未到店就能点餐支付。o2o是永恒不变的话题，而小程序能为微信支付打通o2o的桥梁。

3.小程序+电商

案例

VogueMINI+

  • 主要功能：在优质内容中添加小程序链接，直接进入产品页面，并允许用户购买

分析：

  • 优质内容平台+电商平台，方便用户购买。即看即可买的功能也让用户对产品有更深的了解，激发人的购买欲。优质公众号一直以来的变现方式主要是靠广告收入为主，但是小程序+电商能让众多内容制作者增加了变现的方式，也使本已经趋向平稳发展的公众号平台变得活跃。

总结

小程序对与线下商家、内容制作者具有得天独厚的优势，对于App市场中的低频App产生了巨大的打击。无需安装、无需卸载、触手可及、用完即走的功能使低频app们更加难以生存。小程序的未来发展我个人认为是向线下、电商和低频App中的高频功能整合发展。

三、写在最后

微信小程序作为张小龙对于微信发展所新添加功能，必然有其出现的原因。微信小程序打破了h5页面需要适配不同系统、不同屏幕大小的框架，简便了开发者的开发流程。同时向广大开发者共享微信月活9亿的巨大流量，让小程序开发者们有更多变现、发展的方式。虽然微信小程序并不是新事物，百度等公司也曾经推广过轻应用，但是微信小程序作为移动端的最后一环能很好连接线下，实现o2o，我个人认为微信小程序将是微信支付向线下发展的新开端。而且微信小程序与微信生态的紧密连接也使公众号平台上的优质作者有了更多变现的工具，也使趋于平淡的公众号平台注入新鲜的血液。
